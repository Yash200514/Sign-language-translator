 AI Sign Language to Sentence & Multilingual Translator

 Project Overview

This project is an AI-based Sign Language Interpreter that detects hand gestures using a webcam and converts them into text sentences.
The recognized text can also be translated into multiple languages such as Hindi, Marathi, and Spanish.

The goal of this project is to help bridge communication between people who use sign language and those who do not understand it.

---

 Features

* Real-time hand detection using MediaPipe
* Sign language gesture recognition
* Convert gestures into words and sentences
* Multilingual translation (Hindi, Marathi, Spanish, etc.)
* Webcam-based interaction
* Built using Python and Computer Vision

---

 Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
* Deep Translator

 Installation

1. Clone the repository

```
git clone https://github.com/your-username/sign-language-translator.git
```

2. Navigate to the project folder

```
cd sign-language-translator
```

3. Install dependencies

```
pip install opencv-python mediapipe numpy deep-translator
```

---

 How to Run the Project

Run the following command:

```
python main.py
```

The webcam will start and detect hand gestures.
